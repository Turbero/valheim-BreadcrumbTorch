# Breadcrumb Torch

< Commissioned mod >

Breadcrumb Torch will allow players to spawn a torch of their choice to help them not get lost in big dungeons or areas.

These torches can be broken at any time and won't return any drops.

![](https://i.imgur.com/gV3Nhwe.gif) ![](https://i.imgur.com/fl9PxGN.gif)

* When installed and by default, the hotkey will be "U" and spawned item will be a Green Torch, but they can be changed in .cfg file
* Item and placement is server-synced to apply to all players in servers and spawn the same item.
* Compatible with both dedicated servers and solo runs

## Usage instructions

Configure the spawn item in the .cfg file:

* Height offset against ground
* Placement rules: Anywhere, Only Dungeons or Only Outside
* Prefab name: the name of the item to spawn (presumably a torch but no rules)
